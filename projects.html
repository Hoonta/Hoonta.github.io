Accumulator Management System (AMS) -In Progress

The Accumulator Management System manages all of the electronics inside of the battery. It is responsible for checking the health of the battery, opening and closing the contactors, precharging the Tractive system, and more. Some key features of this project include SPI communication, driving MOSFETS, HV Isolation on PCBs, and CAN communication.

New HV Battery - In Progress

For the 2021 Competition Season, the team is developing a new HV Battery using Li-Ion 18650 cells. This project taught me HV battery design, HV component selection, and interdisciplinary collaboration with Mechanical Engineers.

PedalBoard

The PedalBoard was my first project for the team. This is a board that handles the brake pressure sensors and the throttle position sensors that are located in the front of the car. This project taught me about analog signals, Op-Amps, and the CAN bus.

MasterBlaster

The "MasterBlaster" is the name of the car's control unit, like a car's ECU. This board has 3 CAN lines and communicates with the DAQ system, Motor Controller, and the rest of the car.
